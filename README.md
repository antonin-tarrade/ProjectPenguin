## 0. C'est votre tour !

### Avant de commencer

git clone git@git.inpt.fr:7fault/2022-2023/pass_the_game_2023.git  
git config --local core.autocrlf false (a vérifier pour linux)  
git checkout -b *votre_nom*  
git push --set-upstream origin *votre_nom*
